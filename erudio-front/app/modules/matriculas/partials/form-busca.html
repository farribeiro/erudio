Matrícula

Definição:
Matrículas e enturmações consiste no local destinado para registro de um discente em uma unidade escolar e a definição de turma do mesmo.

Instruções Rápidas:
Pesquise por nome de aluno ou matrícula para localizar o aluno que deseja matricular, clique no botão + para matricular um aluno;
Clique sobre o código de um aluno para visualizar informações de uma matricula.
Clique no balão correspondente na coluna 'opções' para realizar as operações de enturmações, movimentações e informações de um aluno.

Definição:
ex: Escolas, Universidades, Secretarias de Educação, etc...

Dica:
Clique sobre o item desejado para ver as opções disponíveis para ele.
